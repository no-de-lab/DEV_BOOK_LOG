그렇게 할 수 있는 이유가 무엇인가? 
JWT 서버가 인증 서명만 확인한다. 

서버만의 decode 방식이 따로 있기때문에 
디지털 서명만 체크하는식

header, payload, 서명 
jwt 의 서명은 header + payload 암호화로 나온다. 

세션 기반 인증 -> 서버에 부화를 주는 문제가 생긴다, 간편화 한것이 토큰 방식
토큰 기반 인증 -> 토큰 유효한지 아닌지만 알면 된다.
